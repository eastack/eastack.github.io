    classpath {
        downloadJavadoc = true

idea {
    module {
        downloadJavadoc = true
        downloadSources = true
    }
}
gradle cleanIdea idea

显示gradle使用的仓库

添加以下task到build.gradle
task showRepos {
    doLast {
        repositories.each {
            println "repository: ${it.name} ('${it.url}')"
        }
    }
}
执行下列命令显示使用的仓库
gradle -q showRepos
