= EASTACK.ME

Personal Notes.

.Clone the repository
[source, bash]
----
git clone https://github.com/eastack/eastack.github.io
----

.Generate static pages
[source, bash]
----
cd eastack.github.io
./asciidoctor
----

.Start a server
[source, bash]
----
cd public
python -m http.server
----

.Open in browser
[source, bash]
----
xdg-open http://localhost:8000
----
